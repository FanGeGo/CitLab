# CitLab
论文引用实验室，通过对理智学者的仿真，构建纯粹引文网络，通过添加影响因素研究个因素对引用的影响。

本项目主要包括以下内容:

    1. 个人引用行为仿真
    2. 纯粹引文网络形成仿真
    3. 因素分析

本项目将对各个任务进行建模与可视化，将引用的过程从个体与整体，微观与宏观两个不同的方面进行分析和验证。
